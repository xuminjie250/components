📖 使用说明

引入文件

<!-- 1. 引入样式 --> <link rel="stylesheet" href="DateTimePicker.css"> <!-- 2. 引入脚本 --> <script src="DateTimePicker.js"></script>

HTML 结构

<!-- 只需一个普通的 input --> <input type="datetime-local" id="myPicker">

方式一：手动实例化

var picker = new DateTimePicker('#myPicker', { format: 'YYYY-MM-DD HH:mm', // 显示格式 placeholder: '请选择日期时间', // 占位文字 minuteStep: 5, // 分钟步长 (1/5/10/15/30) onChange: function(value, instance) { // 值变更回调 console.log(value); } });

方式二：批量自动初始化

// 自动扫描页面中所有 input[type="datetime-local"] 并初始化 var pickers = DateTimePicker.initAll();

公开 API

picker.getValue() // 获取格式化字符串 "2025-06-15 09:30" picker.getDate() // 获取 Date 对象 picker.setValue(val) // 设置值 (支持字符串/Date/时间戳) picker.setDate(date) // 设置 Date 对象 (链式调用) picker.clear() // 清空 (链式调用) picker.open() // 编程打开面板 (链式调用) picker.close() // 关闭面板 (链式调用) picker.getElement() // 获取原始 input 元素 picker.getWrapper() // 获取最外层包裹元素 picker.destroy() // 销毁实例，还原原始 input

交互说明

// 支持以下交互方式： // 1. 点击输入框 → 弹出日期时间选择面板 // 2. 点击日期区域 → 弹出日历面板 (支持日/月/年三级切换) // 3. 点击时/分数值 → 弹出滚动选择列表 // 4. 双击时/分数值 → 进入键盘直接编辑模式 // 5. 点击"今天" → 快速跳转到今天 // 6. 点击"清除" → 清空已选日期 // 7. 点击面板外部 → 自动关闭
